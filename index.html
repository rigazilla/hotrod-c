hotrod-c
A Reference Implementation in plain C for Hotrod protocol 2.8+

Introduction

This project aims to provide a C library that speeds up Infinispan Hotrod clients development and is:

  • easy to understand and extends;
  • easy to use and install;
  • easy to translate in other languages.

Implementing in other languages

C code is written as plain as possible, it should be quite easy to understand and translate.

To avoid having to deal with system details the transport layer is leaved to the user, who will have to provide functions for read/write bytes from/to the stream:

  • a streamReader function which read bytes from the stream gived a preallocated buffer and the number of bytes to be read.
  • a ‘streamWriter’ function which writes bytes to the stream gived a data buffer and its length. Both the functions require an opaque pointer to a context, that can be used during the execution. Context typically will contain the socket identifier and the io status.

Implementation roadmap should go through:

  • how read n bytes see readBytes()
  • how to read 1 byte see readByte()
  • VInt see readVInt()
  • VLong see readVLong()
  • responseHeader see readResponseHeader()
  • topology see readNewTopology()
  • mediaType see readMediaType()
